# From Ashes to Crown

A medieval fantasy roleplay game where you rebuild a fallen estate using strategic choices, modern knowledge, and relationships.

## Features

- **Choice-driven narrative**: Every decision shapes your story and the fate of Embervale
- **Resource management**: Balance gold, food, wood, stone, and time to survive and thrive
- **Character relationships**: Build trust and loyalty with NPCs through meaningful interactions
- **Stat progression**: Develop your character through charisma, intelligence, strength, leadership, and cunning
- **Hidden knowledge**: Unlock modern skills and ancient mysteries
- **Permanent consequences**: Your choices matter and have lasting effects

## Getting Started

### Requirements
- Node.js 16+
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The game will open at http://localhost:3000

### Building for Production

```bash
npm run build
```

## How to Play

1. Read the story and situation
2. Choose your action carefully - each choice has consequences
3. Manage your estate's resources
4. Build relationships with characters
5. Discover ancient mysteries hidden beneath Embervale
6. Climb from exile to nobility

## Saving Your Game

Click the **📤 Save** button in the top right to export your game state as a JSON file.

## Architecture

- **React-based UI** with custom styling
- **State management** using React hooks
- **Event system** for narrative branching
- **Effect system** for applying choices to game state

## Content

- 100+ story events
- 10 unique characters with relationship stats
- 18 buildings to construct
- Dynamic calendar system with 16-month years
- Skill unlocking through gameplay
- Achievement system
